# SkyWorld One - Design Style Guide

## Design Philosophy

**Visual Language**: Executive, disciplined, global, and trustworthy â€” appealing to consulting and trade clients alike. Inspired by premium financial services aesthetic with a focus on clarity, sophistication, and professional authority.

**Color Palette**:
- Primary: Deep Navy (#1a2332) - Authority and trust
- Secondary: Warm Gold (#c9a96e) - Premium accent and highlights  
- Neutral Base: Pure White (#ffffff) - Clean foundation
- Text Gray: Charcoal (#2d3748) - High contrast readability
- Light Gray: Cool Gray (#f7fafc) - Subtle backgrounds and cards

**Typography**:
- **Display Font**: Playfair Display (serif) - For headings and hero text
  - Elegant, editorial quality with strong personality
  - Used for: H1, H2, hero statements, section headers
- **Body Font**: Inter (sans-serif) - For body text and UI elements
  - Modern, highly legible, professional
  - Used for: Paragraphs, navigation, buttons, forms

**Font Hierarchy**:
- Hero: 4rem (64px) Playfair Display, bold
- H1: 3rem (48px) Playfair Display, semibold  
- H2: 2rem (32px) Playfair Display, medium
- H3: 1.5rem (24px) Inter, semibold
- Body: 1rem (16px) Inter, regular
- Small: 0.875rem (14px) Inter, regular

## Visual Effects

**Animation Library**: Anime.js for smooth, professional animations
**Text Effects**: 
- Gradient text animation on hero headings
- Typewriter effect for key messaging
- Subtle color cycling on accent elements

**Background Effects**:
- Clean white base with subtle geometric patterns
- Animated network connections in hero section
- Soft parallax scrolling on decorative elements

**Interactive Elements**:
- 3D tilt hover effects on cards
- Smooth scale transitions (1.03x on hover)
- Shadow depth changes for elevation feedback
- Color morphing on buttons and links

**Image Treatment**:
- High-contrast, professional photography
- Subtle overlay effects for text readability
- Consistent aspect ratios and spacing
- Premium editorial style compositions

## Component Styling

**Cards**:
- Border radius: 12px for modern softness
- Shadow: Subtle elevation with warm undertones
- Padding: Generous whitespace for premium feel
- Hover: Lift effect with increased shadow

**Buttons**:
- Primary: Deep navy background, gold accent border
- Secondary: White background, navy border
- Border radius: 8px for approachable feel
- Hover: Color inversion with smooth transition

**Navigation**:
- Sticky header with subtle backdrop blur
- Clean typography with adequate spacing
- Dropdown menus with smooth slide animations
- Mobile-first responsive behavior

**Forms**:
- Clean input styling with focus states
- Real-time validation feedback
- Consistent spacing and alignment
- Professional error messaging

## Layout Principles

**Grid System**: 
- 12-column responsive grid
- Consistent gutters (24px desktop, 16px mobile)
- Flexible breakpoints for all screen sizes

**Spacing**:
- Base unit: 8px for consistent rhythm
- Section padding: 80px vertical on desktop
- Card spacing: 32px between elements
- Text line-height: 1.6 for optimal readability

**Responsive Design**:
- Mobile-first approach
- Breakpoints: 640px, 768px, 1024px, 1280px
- Fluid typography scaling
- Touch-friendly interactive elements

## Brand Elements

**Logo Treatment**: Clean, minimal wordmark in navy
**Iconography**: Simple line icons in consistent style
**Photography**: Professional, high-quality business imagery
**Illustrations**: Minimal geometric shapes and patterns

This design system creates a sophisticated, trustworthy brand presence that appeals to executive-level clients while maintaining modern usability and accessibility standards.
